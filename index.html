What this is

Logica is a portfolio of interactive decision systems. Each tool tackles a complex problem — the kind that usually lives inside decks, frameworks, or weeks of consulting analysis — and turns it into a system you can actually run.

Every tool follows the same logic: explicit inputs, structured rules, and clear outputs. The format varies, but the principle stays the same: encode analytical judgment into something anyone can operate.
